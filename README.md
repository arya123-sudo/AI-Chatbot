# 🤖 AI Learning Assistant Chatbot

An intelligent AI-powered chatbot developed using Python, NLP, TF-IDF, Cosine Similarity, Streamlit, and Gemini AI integration.

The chatbot can answer AI-related questions, interact using voice, analyze PDF notes, and provide intelligent responses through Natural Language Processing techniques.

---

# 🚀 Features

* AI & Machine Learning Question Answering
* NLP-based Text Processing
* Intent Detection System
* Conversation Memory
* Voice Assistant Support
* PDF Question Answering
* Streamlit Web Interface

* Intelligent Response Generation

# 💡 How It Works

1. User enters a question
2. NLP preprocessing is applied
3. Intent detection checks common queries
4. TF-IDF vectorization converts text into vectors
5. Cosine similarity finds the best matching response
6. Gemini AI generates intelligent responses
